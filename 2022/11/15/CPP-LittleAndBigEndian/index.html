LSB(Least Significant BIt) and MSB(Most Significant Bit)

LSB refers to the right most bit of a binary number, representing the smallest contribution to the overall number. MSB on the other hand, holding the largest value or determining the sign in signed integers.

Little Endian and Big Endian

Each address identifies a single byte in memory. If a data object takes multiple contiguous bytes in memory, the memory address of this object is defined as the lowest(smallest number) address of all bytes this object contains.
Endianness specifies the order in which bytes of a word are stored in memory.

Little-endian computer

The least significant byte of a word is stored at the smallest memory address. The most significant byte at the largest.

Big-endian computer

Stores the bytes in the opposite order.

  • Byte is the smallest part of difference. Big and Little endian is just need to change the order of Bytes, the bit inside the Bytes no need to change any.
