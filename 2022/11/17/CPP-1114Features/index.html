Strongly typed enums

Before cpp11 enums are defined globally in their namespaces to avoid name collisions, developers tend to add prefixed to the enum names to achieve a uniquess for them

To turn an enum into a strongly typed enum you just have to add a struct keyword.
It is only allowed to use the strongly typed enums with their type specifiers, which guarantees their uniqueness

Besides that, you can define the base type for enums, like here an unsigned 8-bit integer, which allows to save some memory
