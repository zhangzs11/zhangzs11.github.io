Cache Memory

Cache memory is not a computer’s primary memory, nor is it even permanent. But it’s quite important to performance. Cache memory is a small and fast temporary storage area. Efficient, easy, and near-instant data retrieval.
Cache memory hold frequently-accessed data for computer processes. Cache memory, which uses high speed SRAM, can be up to 10 to 100 times faster than the main memory,which uses DRAM.

Three cache levels

Three cache levels describe the chip’s closeness and accessibility to the microprocessor.
An L1 cache (AKA primary cache), is extremely fast and small, and usually embedded in the CPU itself as a CPU cache
An L2 cache (AKA second cache), holds more than L1 cache. It can be embedded like an L1 or be on a separate chip with a high speed alternative system bus that connects the cache and the CPU
AN L3 cache, is specialized memory developed to improve the performance of L1 and L2 caches. While L3 cache is significantly slower that L1 and L2. It is still twice as fast as DRAM.
