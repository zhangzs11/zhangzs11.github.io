How to Control the Game

  • Spacebar: Press the spacebar to accelerate the color change animation in the demo.

Assignment 02 Description:

  • Encapsulate a Mesh class to manage vertex data, and an Effect class to manage shaders and render states. Design and implement platform-independent interfaces for initialization, cleanup, Mesh and Effect, as well as binding Effect and rendering Mesh. Implement these on both OpenGL and D3D11 platforms.

  • Learn to use GPU capture tools to debug and observe graphics programs; I used RenderDoc.

  • Observe the current code structure and consider which platform-dependent code can be encapsulated using the same interface to make it platform-independent and improve code quality.

Observing Code

The remaining differences:

  • Creating and binding buffers to the pipeline.

  • Creating and binding target views to the pipeline.

  • Clearing the rendered texture objects.

  • Submitting the rendering results.

Ways to use platform-independent code:

  • Implement different .cpp files for each platform while sharing the same interface across platforms.

  • Use macro directives to dynamically switch between platform-specific implementations based on the platform being compiled.
