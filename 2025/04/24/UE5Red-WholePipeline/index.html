关键流程描述

PrePass

延迟渲染总是从PrePass开始，这个阶段，顶点着色器绘制geometry并且计算scene的深度值。
如果开启了Nanite geometry，开始使用NaniteVisBuffer。Nanite surface在这个阶段绘制。获得了Nanite geometry的深度值之后，会和之前计算的场景的深度值合并。

LumenSceneUpdate

引擎收集场景数据信息，更新Lumen关于scene的内部表示。（包括 the distance field, voxel{体素} structure, surface cache , etc）

BasePass

像素着色器工作，GBuffer 写入

DiffuseIndirectAndAO

添加soft global illumination, 和 subtle{微小的} shaodws , 增强场景真实度

ShadowDepth

首先剔除object，只有产生阴影并且light能照到的object被渲染depth，然后scene被分成多个VSM pages然后upload。最后动态物体的阴影会通过这些VSM进行叠加

Lights

计算光照，首先计算光源对可见几何体的影响，然后结合材质数据（法线，roughness，etc），计算最终光照结果（延迟渲染高效处理多个动态光源）

SingleLayerWater

单层水通道使用基于物理的模型渲染逼真的水面。它在统一的单一图层中模拟反射、折射和光吸收，从而实现与灯光和环境的自然互动。此通道集成到照明和后期处理流程中，以确保水面与场景的其余部分无缝融合

SkyAtmosphere

天空大气模拟阳光穿过大气层的散射，从而呈现逼真的天空和地平线。指数高度雾添加了随高度消散的地面雾，增强了深度和氛围。场景体积云合成将 3D 云集成到最终图像中，并将它们自然地融合在光照环境中，以保持深度、柔和度和光照一致性

Translucency

半透明阶段负责处理透明和半透明材质的渲染。根据材质与其他效果（例如失真、景深或运动模糊）的相互作用，该阶段分为多个渲染通道。为了提高效率，这些渲染通道并行渲染，并经过精心排序，以便与不透明场景完美融合。

PostProcessing

后期处理阶段通过模拟摄像机和人眼感知世界的方式，对最终图像进行细化。它通过动态曝光、基于运动的模糊、基于深度的聚焦以及基于光线的伪影等效果，增强真实感和视觉质量。此阶段还包括智能升级和时间数据混合，以提高帧间的清晰度和稳定性，最终呈现精致的电影级视觉效果。最后，图片保存在 BackBuffer 中，并在其上绘制一个 UI。

GPU-Driven Culling阶段

场景中Scene Instance 的 Data Structure

为了实现GPU-Driven Culling, UE5需要在GPU side保存 scene的数据表示
我们使用的structure是 GPUScene_InstanceData, which 包含了下面的data

Data Updating

先更新GPU scene数据，然后GPU执行Culling Instance(Frustum culling) 因为游戏逻辑在CPU side执行，所以更新transform等也发生在CPU side， **Input**: Instance Upload Buffer **Output**: GPUScene_InstanceData The new GPU-Driven Culling system has been added to UE5, compared with the old CPU-driven parallel-for-based culling system. 在CPU side，会准备好更新后的data，一个compute shader会更新GPU culling需要的GPUScene_Instance数据基于更新后的info ## Instance Culling

Input: GPUScene_InstanceData
Output: InstanceCulling.VisibleInstanceFlags

GPU的scene更新之后 开始Instance culling. 这个pass获取每个instance的bounding box然后做frustum culling.
InstanceCulling.VisibleInstanceFlags buffer只是一个 packed bit buffer，包含了 instance visibility info作为一个bit

Nanite

主要分成两个stage: the visibility buffer building 以及 the G-buffer emit
